```

│  Chrome Extension│ ◄──────────────►  │  Electron App    │

└─────────────────┘                   └──────────────────┘
```

**Безопасность:**
- WebSocket только на `127.0.0.1` — не слушает сеть
- При старте генерируется секретный токен, записывается в localStorage Electron-а
- Расширение при подключении должно прислать этот токен
- Пароли отправляются расширению ТОЛЬКО при явном клике пользователя на autofill
- Сейф должен быть разблокирован в Electron — иначе расширение не работает

## 1. WebSocket сервер (electron/main/services/websocket.service.ts)

**Порт:** 19823 (фиксированный, уникальный для CipherVault)

**Аутентификация:**
- При старте Electron: генерируем 32-byte токен → записываем в `settings` таблицу
- При подключении расширения: оно присылает `{ action: "auth", token: "..." }`
- Сервер проверяет токен → если ок, добавляет соединение в authenticated set

**Протокол (JSON over WebSocket):**

### Extension → Electron:
```json
{ "action": "auth", "token": "..." }
{ "action": "search", "domain": "github.com" }
{ "action": "get-entry", "id": 42 }
{ "action": "check-status" }
```

### Electron → Extension:
```json
{ "action": "auth-result", "ok": true }
{ "action": "search-result", "entries": [{ "id": 42, "title": "GitHub", "username": "user@email.com" }] }
{ "action": "entry-result", "entry": { "id": 42, "username": "user@email.com", "password": "s3cret" } }
{ "action": "status", "unlocked": true, "alarmMode": false }
```

### Важно:
- Пароль отправляется ТОЛЬКО на `get-entry` (когда пользователь явно выбрал запись)
- На `search` возвращаются ТОЛЬКО `id`, `title`, `username` — БЕЗ пароля
- Если сейф закрыт → расширению возвращается `{ "action": "status", "unlocked": false }`

## 2. Поиск по домену (новая SQL функция)

Текущий `searchEntries` ищет по `display_title`. Нам нужен поиск по URL внутри зашифрованных данных. Проблема: данные зашифрованы.

**Решение:** Получаем все entries для текущего vault, дешифруем, проверяем URL.

```typescript
// entries.service.ts — новая функция
export async function searchEntriesByDomain(domain: string): Promise<Array<{ id: number; title: string; username: string }>> {
  const encKey = getEncryptionKey()
  if (!encKey) return []

  const db = await getDatabase()
  const vaultId = getActiveVaultId()
  const rows = getEntries(db, {}, vaultId) // все entries

  const matches: Array<{ id: number; title: string; username: string }> = []

  for (const row of rows) {
    try {
      const decrypted = decryptJSON<{ url?: string; username?: string }>(
        { iv: row.iv, ciphertext: row.encrypted_data, authTag: row.auth_tag },
        encKey
      )
      if (decrypted.url && matchDomain(decrypted.url, domain)) {
        matches.push({
          id: row.id,
          title: row.display_title,
          username: decrypted.username || '',
        })
      }
    } catch {
      // skip corrupted entries
    }
  }

  return matches
}

function matchDomain(storedUrl: string, currentDomain: string): boolean {
  try {
    const stored = new URL(storedUrl)
    return stored.hostname === currentDomain || stored.hostname.endsWith('.' + currentDomain)
  } catch {
    return storedUrl.toLowerCase().includes(currentDomain.toLowerCase())
  }
}
```

### background.js (Service Worker)
```javascript
let ws = null;
let connected = false;
let token = null;

// Connect to Electron WebSocket server
function connect() {
  if (ws && ws.readyState === WebSocket.OPEN) return;

  ws = new WebSocket('ws://127.0.0.1:19823');

  ws.onopen = () => {
    // Get token from storage and authenticate
    chrome.storage.local.get(['authToken'], (result) => {
      if (result.authToken) {
        ws.send(JSON.stringify({ action: 'auth', token: result.authToken }));
      }
    });
  };

  ws.onmessage = (event) => {
    const msg = JSON.parse(event.data);

    if (msg.action === 'auth-result') {
      if (msg.ok) connected = true;
      return;
    }

    // Forward to content script
    chrome.tabs.query({ active: true, currentWindow: true }, (tabs) => {
      if (tabs[0]) {
        chrome.tabs.sendMessage(tabs[0].id, msg);
      }
    });
  };

  ws.onclose = () => {
    connected = false;
    // Reconnect after 3 seconds
    setTimeout(connect, 3000);
  };

  ws.onerror = () => {
    connected = false;
  };
}

// Listen for messages from content script / popup
chrome.runtime.onMessage.addListener((msg, sender, sendResponse) => {
  if (msg.type === 'FROM_CONTENT') {
    if (ws && ws.readyState === WebSocket.OPEN) {
      ws.send(JSON.stringify(msg.payload));
    }
    return;
  }

  if (msg.type === 'CHECK_STATUS') {
    sendResponse({ connected });
    return;
  }

  if (msg.type === 'SET_TOKEN') {
    token = msg.token;
    chrome.storage.local.set({ authToken: msg.token });
    connect();
    return;
  }
});

// Start connection on install
chrome.runtime.onInstalled.addListener(() => {
  connect();
});

// Also try to connect on startup
connect();
```

### content.js (Form Detection + Autofill)
```javascript
const CV_ICON_CLASS = 'ciphervault-autofill-icon';

// Detect login forms
function findLoginForms() {
  const forms = [];
  const passwordFields = document.querySelectorAll('input[type="password"]');

  for (const pwdField of passwordFields) {
    // Find the form or parent container
    const form = pwdField.closest('form') || pwdField.parentElement;

    // Find username/email field near the password field
    const usernameField = findUsernameField(pwdField, form);

    if (usernameField) {
      forms.push({
        usernameField,
        passwordField: pwdField,
        form,
      });
    }
  }

  return forms;
}

function findUsernameField(pwdField, container) {
  // Strategy 1: Look for input[type="email"] or input[type="text"] with name containing user/email/login
  const candidates = container.querySelectorAll('input[type="email"], input[type="text"], input:not([type])');

  for (const input of candidates) {
    const name = (input.name || '').toLowerCase();
    const id = (input.id || '').toLowerCase();
    const placeholder = (input.placeholder || '').toLowerCase();
    const autocomplete = (input.autocomplete || '').toLowerCase();

    if (name.includes('user') || name.includes('email') || name.includes('login') ||
        id.includes('user') || id.includes('email') || id.includes('login') ||
        placeholder.includes('user') || placeholder.includes('email') || placeholder.includes('login') ||
        autocomplete.includes('username')) {
      return input;
    }
  }

  // Strategy 2: Look for the input just before the password field
  const allInputs = container.querySelectorAll('input');
  for (let i = 0; i < allInputs.length; i++) {
    if (allInputs[i] === pwdField && i > 0) {
      return allInputs[i - 1];
    }
  }

  return null;
}

// Create autofill icon
function createAutofillIcon(entry, usernameField, passwordField) {
  const icon = document.createElement('div');
  icon.className = CV_ICON_CLASS;
  icon.title = `Fill with CipherVault: ${entry.title}`;
  icon.innerHTML = `<svg width="20" height="20" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2"><path d="M12 2l3.09 6.26L22 9.27l-5 4.87 1.18 6.88L12 17.77l-6.18 3.25L7 14.14 2 9.27l6.91-1.01L12 2z"/></svg>`;

  // Position relative to the password field
  const rect = passwordField.getBoundingClientRect();
  icon.style.position = 'absolute';
  icon.style.left = (rect.right + 4) + 'px';
  icon.style.top = (rect.top + (rect.height - 20) / 2 + window.scrollY) + 'px';

  icon.addEventListener('click', (e) => {
    e.preventDefault();
    e.stopPropagation();

    // Request full entry from background
    chrome.runtime.sendMessage({
      type: 'FROM_CONTENT',
      payload: { action: 'get-entry', id: entry.id }
    });
  });

  document.body.appendChild(icon);
  return icon;
}

// Fill the form
function fillForm(username, password) {
  const forms = findLoginForms();
  for (const form of forms) {
    if (form.usernameField) {
      setNativeValue(form.usernameField, username);
    }
    if (form.passwordField) {
      setNativeValue(form.passwordField, password);
    }
  }
}

// Trigger React/Angular change events
function setNativeValue(element, value) {
  const valueSetter = Object.getOwnPropertyDescriptor(window.HTMLInputElement.prototype, 'value').set;
  valueSetter.call(element, value);
  element.dispatchEvent(new Event('input', { bubbles: true }));
  element.dispatchEvent(new Event('change', { bubbles: true }));
}

// Listen for messages from background
chrome.runtime.onMessage.addListener((msg) => {
  if (msg.action === 'search-result') {
    // Remove old icons
    document.querySelectorAll('.' + CV_ICON_CLASS).forEach(el => el.remove());

    // Add autofill icons for each matching entry
    const forms = findLoginForms();
    for (const form of forms) {
      for (const entry of msg.entries) {
        createAutofillIcon(entry, form.usernameField, form.passwordField);
      }
    }
  }

  if (msg.action === 'entry-result' && msg.entry) {
    fillForm(msg.entry.username, msg.entry.password);
    // Remove icons after fill
    document.querySelectorAll('.' + CV_ICON_CLASS).forEach(el => el.remove());
  }

  if (msg.action === 'status' && !msg.unlocked) {
    // Vault is locked — remove all icons
    document.querySelectorAll('.' + CV_ICON_CLASS).forEach(el => el.remove());
  }
});

// On page load, search for matching entries
function onReady() {
  const domain = window.location.hostname;
  chrome.runtime.sendMessage({
    type: 'FROM_CONTENT',
    payload: { action: 'search', domain }
  });
}

if (document.readyState === 'loading') {
  document.addEventListener('DOMContentLoaded', onReady);
} else {
  onReady();
}
```

## 5. Установка расширения (для пользователя)

1. Запустить CipherVault (`npm run dev` или собранный .exe)
2. Открыть `chrome://extensions/` в Chrome
3. Включить "Developer mode"
4. Click "Load unpacked" → выбрать папку `extension/`
5. Расширение подключится к Electron автоматически
