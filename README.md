# CipherVault

Безопасный менеджер паролей с шифрованием AES-256-GCM, построенный на Electron + React + TypeScript.

## Скачать

📥 [GitHub Releases](https://github.com/vipecoder228/CipherVault/releases/latest) — скачай zip, распакуй и запусти `CipherVault.exe`

## Возможности

### Шифрование и безопасность
- **AES-256-GCM** — все данные зашифрованы локально
- **PBKDF2** — 600 000 итераций для защиты от brute-force
- **Дуесс-код** — второй пароль, который открывает пустой vault (защита под угрозой)
- **TOTP 2FA** — двухфакторная аутентификация через Google Authenticator / Authy
- **Проверка утечек** — интеграция с Have I Been Pwned (k-анонимитет)
- **Авто-блокировка** — настраиваемый таймаут + блокировка при сворачивании
- **Ограничение попыток** — блокировка после 10 неудачных попыток

### Browser Extension (autofill)
- **Chrome/Firefox расширение** — autofill паролей на любых сайтах
- **WebSocket** — прямое подключение к Electron-приложению (localhost)
- **Автодетект форм** — расширение само находит логин-пароль поля
- **Иконка на сайтах** — клик и пароль заполнен
- **Безопасно** — пароли не покидают компьютер, только localhost

### Зашифрованный бэкап
- **Экспорт** — весь vault как один `.ciphervault` файл (AES-256-GCM)
- **Импорт** — восстановление на любой машине по паролю
- **Отдельный пароль** — бэкап-пароль не связан с мастер-паролем

### Удобство
- **Генератор паролей** — настраиваемая длина и набор символов
- **Disposable email** — временные почты через mail.tm прямо в приложении
- **Импорт/экспорт** — CSV и JSON (Bitwarden, Chrome, Firefox)
- **Категории** — организация записей с иконками и цветами
- **Избранное** — быстрый доступ к важным записям
- **История изменений** — отслеживание всех изменений записи
- **Режим скрытности** — скрытое окно, глобальная горячая клавиша (Ctrl+Shift+Space)
- **Автоочистка буфера** — копии очищаются через 30 секунд
- **Тёмная и светлая тема**

## Browser Extension — установка

1. Запусти CipherVault и разблокируй vault
2. Открой `chrome://extensions/` в Chrome
3. Включи **Developer mode**
4. Нажми **Load unpacked** → выбери папку `extension/` из репозитория
5. Нажми на иконку расширения → вставь auth token
6. Открой любой сайт с логином → появится иконка CipherVault

## Разработка

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Создание portable-версии
npm run pack

# Создание установщика (требует Developer Mode в Windows)
npm run dist
```

## Безопасность

- Мастер-пароль никогда не хранится — только хеш производного ключа
- Каждая запись зашифрована уникальным IV
- Browser extension работает только через localhost (127.0.0.1)
- Пароли отправляются расширению ТОЛЬКО при явном клике пользователя
- DevTools отключены в продакшене
- Изоляция контекста включена, nodeIntegration отключен
