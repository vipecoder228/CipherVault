# CipherVault

Настольный менеджер паролей с шифрованием AES-256-GCM, построенный на Electron + React + TypeScript.

## Возможности

- **Шифрование AES-256-GCM** — все данные зашифрованы локально вашим мастер-паролем
- **Ключ PBKDF2** — 600 000 итераций для защиты от подбора
- **TOTP 2FA** — двухфакторная аутентификация через приложения-аутентификаторы
- **Дуесс-код** — второй пароль, который открывает пустой vault
- **Проверка утечек** — интеграция с HIBP (k-анонимитет)
- **Импорт/экспорт** — поддержка CSV и JSON
- **Генератор паролей** — настраиваемая длина и набор символов
- **Режим скрытности** — скрытое окно, глобальная горячая клавиша (Ctrl+Shift+Space)
- **Авто-блокировка** — настраиваемый таймаут + блокировка при сворачивании
- **Автоочистка буфера** — копии очищаются через 30 секунд

## Установка

Скачайте установщик `.exe` из [Releases](https://github.com/vipecoder228/CipherVault/releases).

## Разработка

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Создание установщика
npm run dist
```

## Технологии

| Уровень | Технология |
|---------|-----------|
| Десктоп | Electron 33 |
| Интерфейс | React 19 + TypeScript |
| Состояние | Zustand 5 |
| Стили | Tailwind CSS 3 |
| БД | sql.js (SQLite WASM) |
| Крипто | Node.js crypto (AES-256-GCM, PBKDF2) |
| 2FA | otplib + qrcode.react |
| Иконки | Lucide React |
| Сборка | electron-vite + electron-builder |

## Безопасность

- Мастер-пароль никогда не хранится — только хеш производного ключа
- Каждая запись зашифрована уникальным IV
- Ограничение попыток разблокировки (блокировка после 10 неудач)
- DevTools отключены в продакшене
- Изоляция контекста включена, nodeIntegration отключен
